#  CoShare Frontend

Frontend client for **CoShare**, a modern file-sharing UI built with React.

##  Features

- Login & Register
- Profile avatar with user initial
- File upload & download
- Grid-based file cards
- Share links (expire in **12 hours**)
- Toast notifications

##  Tech Stack

React (Vite), Tailwind CSS, Axios, React Router

## Environment Variables (.env):

VITE_API_URL=http://

## Git Clone

```bash
git clone "https://github.com/shamian84/Coshare"
```

##  Run Frontend

```bash
npm install
npm run dev
```
